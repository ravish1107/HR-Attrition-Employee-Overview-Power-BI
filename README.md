# HR Attrition & Employee Overview Dashboard | Power BI

## Project Overview

This project focuses on analyzing employee attrition patterns using the IBM HR Analytics dataset. The goal of this dashboard is to help the HR team understand employee turnover trends across different departments, age groups, job roles, salary groups, and other workforce factors.

The dashboard provides interactive insights that help identify areas with higher employee attrition and support data-driven HR decisions.

---

## Problem Statement

The HR team lacks a visual and interactive way to analyze:

- Which departments have the highest employee turnover
- Which age groups show higher attrition
- How salary levels impact employee retention
- Workforce distribution across departments and roles

This project solves this problem by creating an interactive Power BI dashboard.

---

## Dataset

**Dataset:** IBM HR Analytics Employee Attrition Dataset

Dataset contains:

- 1,470 employee records
- Employee demographics
- Department information
- Job roles
- Salary details
- Attrition status

### Key Columns Used:

- Age
- Department
- Job Role
- Gender
- Monthly Income
- Attrition
- Employee Number

## Dashboard Features

### KPI Cards

- Total Employees
- Employees Left
- Overall Attrition Rate


### Attrition Analysis

- Attrition by Department (Donut Chart)
- Attrition by Age Group (Bar Chart)
- Attrition by Job Role
- Attrition by Gender


### Employee Overview

- Headcount Summary
- Salary Group Analysis
- Interactive Filters


### Interactive Slicers

Users can filter dashboard using:

- Department
- Job Role
- Gender
